Learning Objectives

After completing this unit, you'll be able to:

  • Create point geometry using attribute values.
  • Create line geometry if there are multiple attributes storing coordinates.

Resources

rapid_transit_stations.csv

  1. In a blank workspace, add a CSV reader and browse to the rapid_transit_stations.csv file.
  2. Set the Coord. System to UTM83-10, then click OK. The reader is then added to the canvas.
  3. After the reader is added to the canvas without setting the Attribute Definition in the parameters, add a VertexCreator transformer. The VertexCreator creates a point feature of each record using X/Y/Z or Longitude/Latitude/Elevation values stored in the attributes. Each point feature retains all the attributes of the original non-geometry feature.

  4. Run the workspace and inspect the results.
