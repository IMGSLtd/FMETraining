Learning Objectives

After completing this unit, you'll be able to:

  • Create a line feature by connecting points.
  • Create multiple lines when an attribute value changes.

Resources

rapid_transit_stations.csv

CreateLineFeaturesFromPoints.fmw

Exercise

  1. Open the CreateLineFeaturesFromPoints.fmw workspace. It is the completed workspace from Method 1.
  2. Add a LineBuilder transformer to the canvas connecting it to either the CSV reader (or the VertexCreator depending on the method you chose to get the points)
  3. In the LineBuilder parameters, check Group Processing and set the Group By parameter to LINE. This tells the LineBuilder to stop connecting the points together when the LINE attribute changes. This will result in three different lines in this example. If your lines are not coming out as expected, you may be required to sort by a particular attribute to organize the data by using the Sorter. In FME 2018 or older, the Group By option is not available, instead use Connection Break Attribute.

  4. Run the workspace and inspect the output.
