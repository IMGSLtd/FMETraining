Learning Objectives

After completing this unit, you'll be able to:

  • Identify if a reader will automatically convert attributes to points.
  • Configure a reader to convert attributes to points.

Introduction

Using the reader parameters to create point geometry works well if you are going to be working with spatial data from the start. As of FME 2020, this method occurs automatically as long as the attributes are named correctly, such as latitude, longitude, and elevation, or x, y, and z. It is still possible to use this method in older versions of FME or if your attribute names are different, it just requires a manual step.

Resources

rapid_transit_stations.csv

  1. In a blank workspace, add a CSV reader and browse to the rapid_transit_stations.csv file. 
  2. Set the Coord. System to UTM83-10, then open the parameters. If you are using FME 2020 or newer, confirm that the Attribute Definition has x and y, set to x_coord and y_coord, respectively.
    If you are using an older version of FME or it is not automatically set, switch the Attribute Definition to Manual, then change the X/Y/Z (Longitude/Latitude/Elevation) fields to the x/y/z_coordinate(_coord) data type, click OK until the reader is added to the canvas.
  3. Run the workspace and inspect the results
